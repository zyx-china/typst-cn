# Typst 中文排版技能

Claude Code 技能，为 Typst 中文文档排版提供最佳实践指导。

## 内容

- 中英文字体混排与回退配置
- 段落设置（行距、首行缩进、段间距、两端对齐）
- 标题层级与自动编号
- 目录生成
- 表格与图表的自动编号及交叉引用
- 数学公式（多字母标识符、显示公式、转义问题）
- 常见踩坑点（`@` 转义、`$` 转义、加粗语法等）
- 页面设置、文档元信息、列表缩进

## 安装

```bash
mkdir -p ~/.claude/skills
git clone https://github.com/zyx-china/typst-cn.git ~/.claude/skills/typst-cn
```

## 触发条件

当用户要求用 Typst 排版中文文档、创建 typst 文件、编译 typst、或写中文论文/汇报的 typst 稿子时自动激活。
